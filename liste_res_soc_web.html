Le message

  • France
  • http://www.le-message.org

La France est-elle une démocratie ?

Le message est une proposition concrète qui doit rassembler derrière elle les millions de citoyens dont l’impuissance politique est programmée dans la constitution. En lisant les 6 chapitres de ce site, vous comprendrez pourquoi si vous voulez changer quoi que ce soit dans le fonctionnement actuel de notre société, vous devez faire de ce message votre seule revendication : de son application découlera tout le reste.
